# Media Server Installer

One-command Plex automation stack installer.

Supported services:

- Plex
- Radarr
- Sonarr
- SABnzbd
- Prowlarr
- Bazarr
- Overseerr
- Tdarr
- Tailscale
- Glances
- Watchtower

## Install

curl -fsSL https://raw.githubusercontent.com/FugginOld/media-server-installer/main/install.sh | sudo bash
